# Problem
Given $n$ (where $1 \le n \le 8$ ) pairs of parentheses, write a function to generate all combinations of well-formed parentheses. 

## Example 1
```
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```
## Example 2
```
Input: n = 1
Output: ["()"]
```
